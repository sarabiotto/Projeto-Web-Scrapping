# Monitor de Vagas PCDde concurso - Região Sudeste

Um script de Web Scraping desenvolvido em Python para monitorar e extrair vagas de concursos públicos.

Este projeto foi desenvolvido como requisito para a disciplina de Linguagem de Programação II da Fatec.


# Objetivo

O objetivo deste projeto é automatizar a busca por vagas de concursos públicos no site [PCI Concursos](https://www.pciconcursos.com.br/), filtrando especificamente oportunidades que:
- Estejam localizadas nas cidades de Rio Claro, Piracicaba, Limeira ou Araras ou Região.
